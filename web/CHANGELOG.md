# Changelog — DKMads DMP Web SDK

All notable changes to `@dkmads/dmp-sdk` on npm are documented here.

## [0.1.0] — 2026-06-18

### Added

- Browser DMP client with localStorage `device_pid` persistence
- `getDevicePid()`, `getUserPid()`, `getSharedIdentity()`
- Demographics helpers via `@dkmads/sdk-core`
- Identity contract covered by `tests/integration/sdk-parity/`
